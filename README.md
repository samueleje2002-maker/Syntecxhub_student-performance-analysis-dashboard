 📊 Student Performance Dashboard (Power BI)

 🔍 Project Overview

This project analyzes student performance using Power BI.
It provides insights into student distribution, academic outcomes, and key performance trends across different factors.

 📌 Key Features

 Overall performance: Average scores across all subjects
  Performance breakdown by gender and ethnicity
 Analysis of parental education impact on student outcomes
 Effect of test preparation on student performance
 Top 5 students based on combined scores

 🛠 Tools Used

 Power BI
 Excel (Dataset)

 🚀 Key Insights

 Students who completed test preparation programs consistently performed better across all subjects — showing that small efforts lead to significant improvement
 Noticeable variation in performance across different demographic groups
 Data visualization makes it easier to identify trends and compare outcomes

 📎 How to Use

1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Interact with the dashboard to explore insights


 📖 Extended Description

This dashboard was built to transform raw student data into meaningful insights using interactive visuals.
It highlights how different factors such as preparation, background, and demographics influence student performance.

The project demonstrates my ability to clean data, design dashboards, and communicate insights effectively using Power BI.
